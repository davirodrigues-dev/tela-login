Projeto de interface de login (Front-end) desenvolvido com foco em estilização avançada e layout responsivo. A aplicação simula a tela de acesso ao ambiente EAD da Digital College.

## 🖼️ Visualização

![imagem Tela Login](/images/image.png)

## 📸 Preview

O layout apresenta uma estrutura *Split-Screen* (Tela Dividida):
1.  **Esquerda:** Formulário de autenticação limpo.
2.  **Direita:** Vídeo de background em loop com sobreposição de marca.

## 🚀 Funcionalidades

- [x] **Estrutura Semântica:** HTML5 organizado e acessível.
- [x] **Estilização Avançada:** Uso de CSS Flexbox para alinhamentos.
- [x] **Checkbox Customizado:** Input de "Lembrar meu acesso" totalmente reestilizado via CSS (`::before`, `::after`).
- [x] **Background Video:** Integração de vídeo MP4 com `object-fit: cover` e filtros de cor (overlay).
- [x] **Interatividade:** Efeitos de *hover* em botões e inputs.

## 🛠️ Tecnologias

*   **HTML:** Estruturação da página.
*   **CSS:** Folhas de estilo (Sem frameworks, CSS puro).
*   **Assets:** Ícones e fontes (Google Fonts - Poppins).

## 📂 Estrutura de Pastas

```bash
tela-login/
├── images/
│   ├── background.mp4         # Vídeo lateral
│   ├── fundo.png              # Background do body
│   ├── icone-digital.ico      # Favicon
│   └── logo-academy-preta.png # Logos
├── index.html                 # Arquivo Principal
├── style.css                  # Estilos
└── README.md                  # Documentação
```

## 🔧 Como usar

1.  **Clone o projeto** ou baixe os arquivos `.zip`.
2.  Certifique-se de que a pasta `images` contém todos os recursos visuais.
3.  Abra o arquivo `index.html` em seu navegador.
